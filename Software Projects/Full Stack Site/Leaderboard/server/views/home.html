Recommendation Application

Our MVP idea is an application that allows students, teachers, professors, and faculty to have easier access to recommendation letters. Users that make an account can request letters from one another. The sender is given a form to fill in data about themselves and then sends to the receipient. The receipient of the request can either accept or deny the request depending on if they are willing to write a letter. They can then use a pen tracker to sign the letter they finish writing and then return the pdf file to the sender. This purpose of this application is to be user friendly and nice interface.
